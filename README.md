# node-spider
使用node简单爬取教育网站的相关学校信息

## 运行该程序你需要做以下步骤 
1. 在该文件夹下，在cmd中输入npm i 
```bash
npm i
```
2. 然后就可以正常运行该服务了
```bash
node app1.js
```
